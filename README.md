# Kuis Online Islami
## Deskripsi
Website ini dibuat untuk memenuhi tugas UAS pembuatan website kuis online
menggunakan HTML, CSS, dan JavaScript tanpa framework.

## Fitur
- 5 soal pilihan ganda
- Setiap soal memiliki 4 pilihan jawaban
- Skor otomatis dihitung menggunakan JavaScript
- Jawaban benar bernilai 20
- Lulus jika skor 70 ke atas
- Bisa mengulang kuis

## Alur Program
1. User memilih jawaban
2. Program menghitung total skor
3. Program menampilkan hasil lulus atau tidak lulus
4. User dapat mengulang kuis
